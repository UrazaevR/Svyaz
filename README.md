# Связь

Веб-платформа для проведения онлайн-встреч.  
Проект разработан в рамках дисциплины **«Технологии индустрии 4.0»**.

## О проекте

**Связь** — это веб-приложение для проведения видеоконференций, ориентированное на удобство взаимодействия, простоту использования и доступность для широкого круга пользователей.

Основная идея проекта заключается в создании легковесного и интуитивно понятного инструмента для онлайн-коммуникаций. Платформа ориентирована на работу через браузер и не требует установки отдельного приложения со стороны пользователя.

## Цель проекта

Разработать веб-платформу для видеоконференций, обеспечивающую стабильную аудио- и видеосвязь, удобный интерфейс и доступность для широкого круга пользователей.

## Основные задачи проекта

- разработка минимально жизнеспособного продукта (MVP);
- реализация подключения пользователей к онлайн-встречам;
- обеспечение аудио- и видеосвязи в режиме реального времени;
- разработка удобного пользовательского интерфейса;
- реализация текстового чата внутри конференции;
- добавление демонстрации экрана;
- интеграция интерактивной доски для совместной работы.

## Основные возможности

- регистрация и авторизация пользователей;
- создание комнат для онлайн-встреч;
- подключение участников к комнатам;
- аудио- и видеосвязь в режиме реального времени;
- текстовый чат внутри конференции;
- сохранение истории сообщений;
- демонстрация экрана;
- интеграция интерактивной доски Excalidraw.

## Целевая аудитория

Платформа ориентирована на следующие категории пользователей:

- студенты и преподаватели;
- малые команды и стартапы;
- фрилансеры;
- образовательные проекты и онлайн-курсы с небольшими учебными группами.

- `requirements.txt` — зависимости проекта.

## Установка и запуск
1. Установить Python 3.12 или выше и установите зависимости из файла `requirements.txt`
2. Склонровать репозитори проекта
3. В файле `insightroom-backend/app.py` в последней строке прописать в переменную host доступный вам домен (0.0.0.0 для запуска в локальной сети)
4. Запустить PostgreSQL сервер на своей машине [инструкция по запуску БД](https://www.postgresql.org/docs/current/server-start.html)
5. Создать БД в соответствии с ER диаграммой (`insightroom-backend/ER диаграмма БД.png`)
6. В файле `insightroom-backend/data/main.py` в последней строке прописать ip адрес для подключения к БД, а также имя БД, имя пользователя и пароль от БД
7. Запустить файл `insightroom-backend/app.py`, открыть ссылку, которая будет написана в консоли
